Le seul format de fichier autorisé est le .csv.
Les données du fichier doivent être délimitées par des ;.
Les seuls encodages de fichiers supportés sont UTF-8 et ANSI.
Le fichier doit contenir au maximum 60 patients.
Tout les champs ne sont pas obligatoire, hormis le nom et le prénom du patient.
Lors du traitement, la première ligne du fichier n'est pas prise en compte.
Le fichier doit comporter exactement 17 colonnes.
Les colonnes du fichiers doivent représenter dans l'ordre :
{{ cln.key }} {{ cln.value }}
